## 3. Design decisions

- Canonical JSON is key-sorted and hashed with deterministic SHA-256. Invalid/missing deployment SHA is `LOCAL_UNCONFIGURED`, never a fake commit.
- Migration 0010 is additive; existing rows become `LEGACY_UNVERSIONED`. v1 endpoints reject invented provenance.
- Backtest/robustness v1 responses distinguish runtime model identity from the validated persisted identities of returned snapshot rows.
- Admin auth accepts exact legacy bearer or configured Access service-token credentials. Full rebuild needs the independent exact `FULL_REBUILD` header.
- Typed provider failures pass through the cache loader so they increment the circuit. Stale or failed stress is always exposed as `UNKNOWN`, never stale `NORMAL` guidance.
- Backup has no side effect without `--execute`; production adds exact confirmation. Restore uses only an ephemeral local D1 fixture.
